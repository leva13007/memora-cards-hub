# Sprint 01

## Goals
The primary goal of Sprint 01 is to conduct comprehensive testing of the newly developed UI components in the Storybook environment. This includes verifying the functionality, appearance, and responsiveness of Text components (Light, Medium, Bold), TextLink, H1, H2, and Breadcrumbs components to ensure they meet the specified requirements and are ready for integration into the main application.

---

## Scope

### In Scope
- UI Kit Text components (Light, Medium, Bold)
- UI Kit TextLink component
- UI Kit H1 component
- UI Kit H2 component
- UI Kit Breadcrumbs component

### Out of Scope
- Integration of UI components into the main application.
- Backend functionality or API testing.
- Performance testing.
